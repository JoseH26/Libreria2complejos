LIBRERÍA VECTORES Y MATRICES - NÚMEROS COMPLEJOS

En esta librería se presenta el programa para continuar trabajando con numeros complejo, esta vez aplicados en vectores y matrices.

Pre-requisitos.
Es necesario descargar el lenguaje de programación Python.
Obtener una copia de la librería publicada en el repositorio.

Descargar e Instalar Python.
1. Ir al siguiente link: https://www.python.org/downloads/
2. Hacer clic en el botón “Download Python”
3. Una vez completada la descarga, abra el instalador, presione “siguiente” y aceptar los términos y condiciones de la aplicación.

Instalación.	
Obtener una copia del Proyecto.
1. Dijirse al link del repositorio: git remote add origin https://github.com/JoseH26/Libreria2complejos.git

2. Pulsar el botón “Code” de color verde.
3. Seleccionar la opción “Download ZIP”
4. Descomprimir el .ZIP y ejecutar el archivo .py

Ejecutando las pruebas.
Para realizar pruebas en la librería, abrir python y dar click en "File", luego "open" y seleccionar el archivo “Complejos2JH.py”, una vez allí encontrará todas las operaciones de los numeros complejos en matrices y vectores,
finalmente se da cluck en "Run" para ejecutar el codigo cuantas veces quiera el usuario con distintos valores, el programa ejecutará todas las operaciones en simultaneo imprimiendo sus respectivos resultados.
 
Construido con
Python - IDLE (Python 3.9 64-bit)
